Web Design Style Guide

Color


Color system Explanation Opacity
rgb Red, green, blue color components from 0 - 255 a (alpha/opacity) 0 - 1, 0 is transparent 1 is opaque
hsl Hue: 0 - 360 on color wheel
Saturation: 0% - 100%
Lightness %		 a (alpha/opacity) 0 - 1, 0 is transparent 1 is opaque
hex Colour code preceded by #

Property key-words: color, background-color, opacity.


Font


'Standard' fonts Web fonts
Use the 'font-family' property Google, Adobe
Use links provided in the head. Font-family will be provided

Property key-words: font-family, font-weight
font-style, text-align, text-transform


Links


  • Anchor opening and closing tags, a, make text a hyperlink
    enter href to insert the link address (e.g. webpage)
    use target equals "_blank" to make it open in a new tab

  • We can make images into a link by wrapping the code in anchor tags

  • An id attribute can be used to link to a section on the page, e.g. #top


Pseudo-class Action
:link Normal (not clicked) pre-click
:visited Once clicked
:hover Hover action
:active Clicked

Property key-words: text-decoration (underline)
title (hover text), cursor (pointer)
